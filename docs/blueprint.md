# **App Name**: NeonWallet

## Core Features:

- Login / Register: User authentication using Firebase Authentication.
- Dashboard: Display total balance with animated updates.
- Wallet: List coins in a card format.
- Transactions: Send and receive transactions, saved on Firestore.
- AI powered asset analysis: Uses an AI tool to suggest potential investment options.

## Style Guidelines:

- Primary color: Neon violet (#A855F7) for a futuristic and vibrant feel.
- Background color: Dark background (#111827) to enhance the neon effect.
- Accent color: Neon blue (#3B82F6) to complement the violet and add depth.
- Body and headline font: 'Space Grotesk' for a modern, techy look. Pairs well with a neon aesthetic. Note: currently only Google Fonts are supported.
- Cards with glassmorphism effect for a premium look.
- Rounded borders for a soft and modern UI.
- Use Framer Motion for smooth and engaging animations.
